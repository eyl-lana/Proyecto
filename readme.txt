- FUNCIONALIDADES -

1.- El usuario puede crear una cuenta identificando 'Nombre', 'Apellido, 'Correo electrónico' y 'Contraseña'.
2.- Si el usuario ya tiene una cuenta, solo debe iniciar sesión con el correo electrónico y contraseña que utilizó al momento de crearse una cuenta.
3.- Sin cuenta registrada el usuario solo puede buscar recetas y dietas.
4.- Con cuenta registrada el usuario puede buscar, subir, comentar recetas, clasificar recetas y comentar dietas.
5.- Las recetas serán filtradas según el nivel de dificultad de esta, por lo que se pueden buscar recetas de nivel fácil, nivel medio y nivel difícil.
6.- Al subir una receta se debe agregar una introducción, imagen de la receta resultante, el tiempo de preparación, los ingredientes que necesita, los pasos
    de la receta y el nivel de dificultad.
7.- ¿Por qué comer sano? Interfaz para motivar a la población a vivir una vida sana.
